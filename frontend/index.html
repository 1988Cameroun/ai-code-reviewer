CodeSentinel runs a two-pass LLM evaluation on every submission — designed to catch what single-pass reviews miss.

Pass 1 — Primary Analysis
Evaluates correctness, security vulnerabilities, performance bottlenecks, and scalability across four scored dimensions.
Pass 2 — Meta-Evaluation
A second LLM pass audits the first review — checking for missed issues, inflated scores, and blind spots. AI evaluating AI.
Persistence Layer
All evaluations are logged to SQLite with full score history for trend analysis and auditability.
